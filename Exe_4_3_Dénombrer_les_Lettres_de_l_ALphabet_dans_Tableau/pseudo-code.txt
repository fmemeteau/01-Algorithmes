Lire un texte d'au moins 120 caractères (à contrôler). 
Compter et afficher le nombre d'occurences de chacune des lettres de l'alphabet.

VARIABLES
	text contient le texte à vérifier
	alphabet est un string contenant les 26 lettres de l'alphabet aisi que les accents en lowercase
	alphabetUp est un string contenant les 26 lettres de l'alphabet en uppercase


TRAITEMENT
	SPLIT les ' ', ',', ''', '.' dans le texte et attribuer ce nouveau texte dans text
	POUR CHAQUE lettre dans le texte
		COMPARER la lettre avec l'un des caractères des 2 alphabets
